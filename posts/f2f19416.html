иҝҷйҮҢжәҗз Ғзҡ„жіЁйҮҠжҳҜвҖңи®©beanpostprocessorжңүжңәдјҡиҝ”еӣһд»ЈзҗҶиҖҢдёҚжҳҜзӣ®ж Үbeanе®һдҫӢвҖқпјҢеҰӮжһңжҲҗеҠҹиҝ”еӣһд»ЈзҗҶеҜ№иұЎпјҢеҲҷзӣҙжҺҘиҝ”еӣһпјҢеҗҰеҲҷеҶҚжү§иЎҢdoCreateBeanжқҘеҲӣе»әе®һдҫӢгҖӮд»ҺејҖе§Ӣиҝӣе…ҘеҲ°applyBeanPostProcessorsBeforeInstantiationиҝҷдёӘж–№жі•дёӯпјҢдјҡдҪҶжҳҜеҜ№дәҺAbstractAutoProxyCreatorиҝҷдёӘзұ»жқҘиҜҙпјҢеңЁжү§иЎҢapplyBeanPostProcessorsBeforeInstantiationж–№жі•ж—¶пјҢе®ғе№¶жІЎжңүеҲӣе»әпјҢжүҖд»Ҙе№¶дёҚдјҡжү§иЎҢAbstractAutoProxyCreatorзҡ„applyBeanPostProcessorsBeforeInstantiationж–№жі•е’ҢapplyBeanPostProcessorsAfterInitializationпјҢиҝ”еӣһnull,еңЁеҗҺйқўзҡ„йҖ»иҫ‘дёӯпјҢеҰӮжһңresolveBeforeInstantiationиҝ”еӣһnull,дјҡжү§иЎҢdoCreateBeanж–№жі•пјҢеҲӣе»әAbstractAutoProxyCreator

4.2.3 doCreateBeanеҲӣе»әе®һдҫӢ

еҪ“resolveBeforeInstantiationиҝ”еӣһnullж—¶пјҢжү§иЎҢdoCreateBean
doCreateBeanдё»иҰҒжөҒзЁӢжҳҜпјҡ

  • 1пјүгҖҒеҲӣе»әBeanзҡ„е®һдҫӢ
  • 2пјүгҖҒpopulateBeanпјӣз»ҷbeanзҡ„еҗ„з§ҚеұһжҖ§иөӢеҖј
  • 3пјүгҖҒinitializeBeanпјҡеҲқе§ӢеҢ–beanпјӣ

е®ғзҡ„дё»иҰҒжӯҘйӘӨеңЁдәҺinitializeBeanж–№жі•дёҠ

4.3.2 initializeBeanеҲқе§ӢеҢ–bean

  • 1пјүгҖҒinvokeAwareMethods()пјҡеӨ„зҗҶAwareжҺҘеҸЈзҡ„ж–№жі•еӣһи°ғ
  • 2пјүгҖҒapplyBeanPostProcessorsBeforeInitialization()пјҡеә”з”ЁеҗҺзҪ®еӨ„зҗҶеҷЁзҡ„postProcessBeforeInitializationпјҲпјү
  • 3пјүгҖҒinvokeInitMethods()пјӣжү§иЎҢиҮӘе®ҡд№үзҡ„еҲқе§ӢеҢ–ж–№жі•
  • 4пјүгҖҒapplyBeanPostProcessorsAfterInitialization()пјӣжү§иЎҢеҗҺзҪ®еӨ„зҗҶеҷЁзҡ„postProcessAfterInitializationпјҲпјү

5 finishBeanFactoryInitialization

з¬¬еӣӣз« иҠӮдё»иҰҒжҳҜд»Ӣз»ҚеҲӣе»әе’ҢжіЁеҶҢAnnotationAwareAspectJAutoProxyCreatorеҗҺзҪ®еӨ„зҗҶеҷЁзҡ„иҝҮзЁӢ
иҖҢfinishBeanFactoryInitialization(beanFactory)жҳҜе®ҢжҲҗBeanFactoryеҲқе§ӢеҢ–е·ҘдҪңгҖҒеҲӣе»әеү©дҪҷзҡ„йқһжҮ’еҠ иҪҪеҚ•е®һдҫӢbeanпјҢд№ҹе°ұжҳҜжҲ‘д»¬зҡ„дёҡеҠЎbeanпјҡMathCalculatorзҡ„еҲӣе»әиҝҮзЁӢжҳҜеңЁиҝҷдёҖжӯҘе®ҢжҲҗзҡ„

beanFactory.preInstantiateSingletons()

еңЁиҝҷдёӘж–№жі•йҮҢйқўе…іжіЁжңҖеҗҺдёҖдёӘж–№жі•пјҢbeanFactory.preInstantiateSingletons()зҡ„дҪңз”Ёе°ұжҳҜеҲӣе»әеү©дҪҷзҡ„йқһжҮ’еҠ иҪҪеҚ•е®һдҫӢbeanпјҢеҰӮдёӢеӣҫжүҖзӨәпјҡ


жҲ‘д»¬зҡ„дёҡеҠЎcalculatorпјҢLogAspectsе°ұжҳҜеңЁиҝҷйҮҢеҲӣе»әзҡ„пјҢдјҡи°ғз”ЁзҲ¶зұ»зҡ„AbstractBeanFactoryзҡ„doGetBeanж–№жі•пјҢжү§иЎҢжөҒзЁӢе’Ңз« иҠӮ4.2зҡ„getBeanж–№жі•жҳҜдёҖдёӘжөҒзЁӢ

дҪҶжҳҜеҲӣе»әcalculatorпјҢLogAspectsе’ҢеҲӣе»әAnnotationAwareAspectJAutoProxyCreatorжҳҜжңүзӮ№дёҚеҗҢзҡ„пјҢеңЁ4.2.2з« иҠӮпјҢжҲ‘д»¬жңүжҸҗеҲ°пјҢеҲӣе»әbeanж—¶пјҢresolveBeforeInstantiation(beanName, mbdToUse)ж–№жі•дјҡе°қиҜ•иҝ”еӣһе®ғзҡ„д»ЈзҗҶеҜ№иұЎиҝ”еӣһпјҢеҗҰеҲҷжү§иЎҢdoCreateBeanеҲӣе»әbeanеҜ№иұЎпјҢеӣ дёәAnnotationAwareAspectJAutoProxyCreatorе·Із»ҸеңЁз¬¬4з« иҠӮзҡ„registerBeanPostProcessorsж–№жі•дёӯжүҖеҲӣе»әпјҢжүҖд»ҘеңЁжҜҸдёҖдёӘbeanеҲӣе»әд№ӢеүҚпјҢйғҪдјҡи°ғз”ЁAnnotationAwareAspectJAutoProxyCreatorзҡ„postProcessBeforeInstantiation()ж–№жі•е’ҢpostProcessAfterInitializationж–№жі•

5.1 postProcessBeforeInstantiation()

AnnotationAwareAspectJAutoProxyCreatorзҡ„зҲ¶зұ»AbstractAutoProxyCreatorзҡ„postProcessBeforeInstantiationж–№жі•еҰӮдёӢпјҡ


иҝҷдёӘж–№жі•д»ҘMathCalculatorе’ҢLogAspectзҡ„еҲӣе»әдёәдҫӢеӯҗпјҡ

  • 1пјүгҖҒеҲӨж–ӯеҪ“еүҚbeanжҳҜеҗҰеңЁadvisedBeansдёӯпјҲдҝқеӯҳдәҶжүҖжңүйңҖиҰҒеўһејәbeanпјүпјҢз¬¬дёҖж¬ЎеҲӣе»әиӮҜе®ҡжҳҜжІЎжңүзҡ„
  • 2пјүгҖҒеҲӨж–ӯеҪ“еүҚbeanжҳҜеҗҰжҳҜеҹәзЎҖзұ»еһӢзҡ„AdviceгҖҒPointcutгҖҒAdvisorгҖҒAopInfrastructureBeanпјҢ жҲ–иҖ…жҳҜеҗҰжҳҜеҲҮйқўпјҲ@Aspectпјү

  • 3пјүгҖҒжҳҜеҗҰйңҖиҰҒи·іиҝҮ
    • 1пјүгҖҒиҺ·еҸ–еҖҷйҖүзҡ„еўһејәеҷЁпјҲеҲҮйқўйҮҢйқўзҡ„йҖҡзҹҘж–№жі•пјүгҖҗList candidateAdvisorsгҖ‘
       жҜҸдёҖдёӘе°ҒиЈ…зҡ„йҖҡзҹҘж–№жі•зҡ„еўһејәеҷЁжҳҜ InstantiationModelAwarePointcutAdvisorпјӣ
 еҲӨж–ӯжҜҸдёҖдёӘеўһејәеҷЁжҳҜеҗҰжҳҜ AspectJPointcutAdvisor зұ»еһӢзҡ„пјӣиҝ”еӣһtrue
    • 2пјүгҖҒж°ёиҝңиҝ”еӣһfalse

5.2 postProcessAfterInitialization()

AnnotationAwareAspectJAutoProxyCreatorзҡ„зҲ¶зұ»AbstractAutoProxyCreatorзҡ„postProcessAfterInitializationж–№жі•жҳҜеңЁ
з« иҠӮ4.3.2 initializeBeanеҲқе§ӢеҢ–beanзҡ„applyBeanPostProcessorsAfterInitializationи°ғз”Ёзҡ„

postProcessAfterInitialization()д»Јз ҒеҰӮдёӢпјҡ

еҸҜд»ҘзңӢеҮәпјҢиҝ”еӣһдәҶдёҖдёӘwrapIfNecessary(bean, beanName, cacheKey);//еҰӮжһңйңҖиҰҒеҢ…иЈ…
wrapIfNecessary(bean, beanName, cacheKey);д»Јз ҒеҰӮдёӢ

еүҚйқўзҡ„дёүдёӘifеңЁ5.1е·Із»ҸжңүиҝҮд»Ӣз»ҚпјҢдё»иҰҒзңӢObject[] specificInterceptors = getAdvicesAndAdvisorsForBean(bean.getClass(), beanName, null)пјҢиҺ·еҸ–еҪ“еүҚbeanзҡ„жүҖжңүеўһејәеҷЁпјҲйҖҡзҹҘж–№жі•пјүпјҢиҝҷйҮҢзҡ„еҪ“еүҚbeanд№ҹе°ұжҳҜMathCalculatorпјҢиҝҷдёӘж–№жі•еҒҡдәҶеҰӮдёӢеҮ дёӘжӯҘйӘӨ

  • 1гҖҒжүҫеҲ°еҖҷйҖүзҡ„жүҖжңүзҡ„еўһејәеҷЁпјҲжүҫе“ӘдәӣйҖҡзҹҘж–№жі•жҳҜйңҖиҰҒеҲҮе…ҘеҪ“еүҚbeanж–№жі•зҡ„пјү
  • 2гҖҒиҺ·еҸ–еҲ°иғҪеңЁbeanдҪҝз”Ёзҡ„еўһејәеҷЁгҖӮ
  • 3гҖҒз»ҷеўһејәеҷЁжҺ’еәҸ
  • 4гҖҒдҝқеӯҳеҪ“еүҚbeanеңЁadvisedBeansдёӯпјӣ

еҰӮжһңеҪ“еүҚbeanйңҖиҰҒеўһејәпјҢдјҡеҲӣе»әеҪ“еүҚbeanзҡ„д»ЈзҗҶеҜ№иұЎпјҡObject proxy = createProxy(bean.getClass(), beanName, specificInterceptors, new SingletonTargetSource(bean));
protected Object createProxy(Class<?> beanClass, @Nullable String beanName,
@Nullable Object[] specificInterceptors, TargetSource targetSource) {

    if (this.beanFactory instanceof ConfigurableListableBeanFactory) {
        AutoProxyUtils.exposeTargetClass((ConfigurableListableBeanFactory) this.beanFactory, beanName, beanClass);
    }

    ProxyFactory proxyFactory = new ProxyFactory();
    proxyFactory.copyFrom(this);

    if (!proxyFactory.isProxyTargetClass()) {
        if (shouldProxyTargetClass(beanClass, beanName)) {
            proxyFactory.setProxyTargetClass(true);
        }
        else {
            evaluateProxyInterfaces(beanClass, proxyFactory);
        }
    }
    // 1пјүгҖҒиҺ·еҸ–жүҖжңүеўһејәеҷЁпјҲйҖҡзҹҘж–№жі•пјү
    Advisor[] advisors = buildAdvisors(beanName, specificInterceptors);
    2пјүгҖҒдҝқеӯҳеҲ°proxyFactory
    proxyFactory.addAdvisors(advisors);
    proxyFactory.setTargetSource(targetSource);
    customizeProxyFactory(proxyFactory);

    proxyFactory.setFrozen(this.freezeProxy);
    if (advisorsPreFiltered()) {
        proxyFactory.setPreFiltered(true);
    }
    3пјүгҖҒеҲӣе»әд»ЈзҗҶеҜ№иұЎпјҡSpringиҮӘеҠЁеҶіе®ҡ
    return proxyFactory.getProxy(getProxyClassLoader());
}

  • 2пјүгҖҒдҝқеӯҳеҲ°proxyFactory
  • 3пјүгҖҒеҲӣе»әд»ЈзҗҶеҜ№иұЎпјҡSpringиҮӘеҠЁеҶіе®ҡ
